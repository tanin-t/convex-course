## 🪞 Reflect (รวมทั้งคอร์ส)
**คำถาม 5 ข้อสุดท้าย:**
1. **Mindset shift** — Course นี้เปลี่ยนวิธีฉันคิดเรื่อง code ยังไง
2. **Skill gain** — Skill ที่พัฒนามากที่สุดของฉันคือ ... — เพราะ session ไหน
3. **AI relationship** — ความสัมพันธ์ของฉันกับ AI ก่อนคอร์ส vs. หลัง — ต่างยังไง
4. **Gap** — สิ่งที่ฉันรู้ว่ายังไม่รู้คือ ...
5. **Next** — ฉันจะเรียนต่อด้วยตัวเองยังไง — เริ่มจากไหน

## 🎤 Assessment (Present 15 นาทีต่อคน)
1. **Demo** — แสดง project
2. **Concept** — concept ที่สำคัญที่สุด อธิบายให้คนที่ไม่อยู่ห้องนี้เข้าใจ
3. **Hardest bug** — เล่า bug ที่ยากที่สุด + วิธี debug
4. **Learning journey** — ฉันเรียนรู้อะไรเกี่ยวกับ "ตัวฉันเอง" ในฐานะ developer
5. **Next steps** — สิ่งที่อยากเรียนต่อ + แผนการ
